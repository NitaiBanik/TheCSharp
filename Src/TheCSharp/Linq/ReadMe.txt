# Introduced from .NET 3.5 C# 3.0

# Main task is generate queries to retrive data from the data source (Linq compatable data source).

# Provide common syntax for quering all datasources.

# Linq takes deffered execution.

# Many types of LINQ - Linq to objects, Linq to Xml, Linq to EntityFramework,
Linq to Dataset, Linq to Sql (DLinq)

# Many query operators - Select, From, Where, Orderby, Join, Groupby etc for supporing
  filtering, sorting, ordering grouping

# Provide full type checking at compile time.

# Linq supports query expression, Implicitly typed variables, Object and collection initializers, 
Anonymous types, Extension methods, and Lambda expressions.

# Linq query can be reused and it increase code readability.

# Linq consist with 3 layers, top layer is language extension then .NET framework components and then datasource.

# All Linq components are under System.Linq namespace
