- dynamic is a type that can avoid compile time type checking
- It resolve at runtime. So all exception is thrown at runtime. 
- Dynamic type variable is created using "dynamic" keyword.
- Visual studio does not provide any intellesence.
- It is different than "var"
- It is introduce from c# version 4
- It is diffenet from object type as well. Object type has boxing and unboxing. So it is slow. But dynamic is fast.
- in dynamic, any incorrect casting can create runtime error.
